Steps to install GIT:
    1. Download Git from https://git-scm.com/downloads
    2. Adding SSH Key
        https://docs.github.com/en/authentication/connecting-to-github-with-ssh/generating-a-new-ssh-key-and-adding-it-to-the-ssh-agent
    3. Config
            $ git config --global user.name “Jon Rosado”
            $ git config --global user.email “jon.rodado42@gmail.com"
    4. Commands:
        To initialize a empty git: git init
        To check the file change status: git status
        To add files: git add <FileName>
        To commit the files: git commit -m "some text"
        To push the code to remote repo: git push origin branchName
        to check the current branch name: git branch
        to create a new branch: git checkout -b "branchName"
